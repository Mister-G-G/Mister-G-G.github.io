# Mister-G-G Upload-Seite

Dieses Repository enthält eine einfache Flask-Anwendung zum Hochladen von Dateien. Die Dateien werden nach Fächern sortiert im Ordner `uploads` gespeichert.

## Voraussetzungen

- Python 3.12 oder neuer
- Abhängigkeiten aus `requirements.txt`

## Installation

```bash
pip install -r requirements.txt
```

## Anwendung starten

```bash
python app.py
```

Die Anwendung ist anschließend unter `http://localhost:5000` erreichbar.
